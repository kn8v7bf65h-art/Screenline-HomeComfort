# ScreenLine HomeComfort v0.2.1

This release adds HomeComfort-style branding to the Home Assistant integration.

## Changes

- Added a square integration icon in the HomeComfort dark-and-gold style.
- Added a rectangular HomeComfort logo.
- Added separate light and dark mode assets.
- Added HiDPI `@2x` assets.
- Added local Home Assistant brand assets under `custom_components/screenline_homecomfort/brand/`.
- Added repository-level HACS brand assets under `brand/`.
- Updated the integration version to `0.2.1`.

## Compatibility

Local brand assets require Home Assistant 2026.3 or newer. The integration itself retains the cloud login and local WISE Hub support from v0.2.0.
